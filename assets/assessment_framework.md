# 評価フレームワークの図の説明

この図は、生成AIの脱属人化レベルを評価するための4段階のフレームワークを示しています：

**レベル1: 個人依存**
- 少数の先駆者のみが利用
- 非公式なサポート体制
- 標準化されていない利用方法

**レベル2: 部分的統合**
- 部門内での利用拡大
- 基本的なガイドラインの策定
- トレーニングプログラムの開始

**レベル3: 組織的統合**
- 全社的な利用
- 包括的なガイドラインと標準
- 自己学習リソースの充実

**レベル4: 文化的統合**
- 日常業務への完全統合
- 継続的な改善文化
- イノベーションの促進

各レベルはピラミッド型で表示され、上に行くほど高度な統合状態を示しています。
